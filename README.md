# imagelint

## Usage

```
image-lint --max-width 800 "./**/*.png" "./**/*.jpg"
```

## Help

```
Usage of image-lint:
  -markdown
        show errors in markdown list format (default true)
  -max-height int
        max height (default 1080)
  -max-width int
        max width (default 1920)
  -min-height int
        min height (default 1)
  -min-width int
        min width (default 1)

  globs(**) ...
```
